# 🎮 Popkings

**Popkings** — это веселая и ненапряжная инди-игра для Android, созданная на Unity. Простая механика, яркий визуальный стиль и позитивное настроение — идеальный способ расслабиться после долгого дня.

---

## ✨ Особенности игры

- 🎯 **Простой и интуитивный геймплей** — разберетесь за секунды
- 😌 **Без стресса** — игра создана для отдыха и хорошего настроения
- 🎨 **Яркая стилистика** — радует глаз и создает уютную атмосферу
- ⏱️ **Короткие сессии** — можно играть в любую свободную минуту
- 📱 **Оптимизирована для Android** — плавная работа на большинстве устройств

---

## 🕹️ Управление

- **Тап по экрану** — основное действие
- Все взаимодействия интуитивно понятны и не требуют обучения

## 🛠️ Технологии

- **Движок:** Unity 2021.3 LTS (или ваша версия)
- **Платформа:** Android (API Level 21+)
- **Язык:** C#

---

## 📥 Установка

1. Скачайте APK-файл на свой телефон
2. Откройте файл и разрешите установку из неизвестных источников (если потребуется)
3. Установите игру
4. Наслаждайтесь игрой! 🎮
